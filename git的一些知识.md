## git的一些知识

git的工作区是文件夹。分支与工作区没有关系

工作区、**暂存区**、版本库

2.

4.

红色

绿色：放入暂存区之中

5.

git add 加入暂存区

6.

git commit -m "给自己看的备注信息"：**将暂存区的内容提交到当前分支**

暂存区会清空







工作区：仓库的目录。工作区是独立于各个分支的。

暂存区：数据暂时存放的区域，类似于工作区写入版本库前的缓存区。暂存区是独立于各个分支的。
版本库：存放所有已经提交到本地仓库的代码版本

版本结构：树结构，树中每个节点代表一个代码版本。
